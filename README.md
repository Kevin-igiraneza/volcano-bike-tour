# Volcano Bike Tour Website - Launch Instructions

## 1. Prerequisites
- A modern web browser (Chrome, Firefox, Safari, or Edge).
- Your logo image (provided) should be named `logo.png` and placed in the same folder as `index.html`.

## 2. Setup
1. Ensure all three files (`index.html`, `style.css`, `script.js`) are in the same directory.
2. Place your logo file in the same directory and name it `logo.png`.

## 3. How to Run
- **Option A (Simplest):** Just double-click the `index.html` file in your folder. It will open in your default browser.
- **Option B (Recommended for Dev):** Use a local server like "Live Server" extension in VS Code. This ensures all paths and scripts load perfectly.

## 4. Customization
- **Images:** I have used high-quality placeholders from Unsplash. You can replace the `src` attributes in `index.html` with your own photos of the tours and Rwanda.
- **Contact Info:** Open `index.html` and search for "Contact" to update the email or phone number if they change.
- **Prices:** Update the tour card prices directly in the `index.html` file.
- **Logo:** The website is designed to harmonize with your logo colors (Green, Gold, Blue, Black). If you want to tweak specific shades, open `style.css` and change the variables at the top under `:root`.

Enjoy your new professional tourism website!
